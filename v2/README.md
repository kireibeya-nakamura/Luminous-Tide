# Luminous Tide v2 — ゼロからの作り直し

引き継ぎテキストと参照画像だけを元に、既存の `legacy/` を参照せず新規に実装したプロトタイプ。
公開先: https://kireibeya-nakamura.github.io/Luminous-Tide/v2/

## 設計の芯：カメラ深度モデル

水面と水中を別画面として作らず、**カメラ深度 `cam.y` という1つの数字**で繋いだ。

| cam.y | 見えるもの |
|---|---|
| `0` | 水面画面（水平線は画面の40%の高さ） |
| `0 → crossDepth` | 潜降。視点が下がり、水平線が明るい1本の線に集約されながら画面上部へ昇る |
| `crossDepth` | 潜り切った状態。線は上部46pxに「水面断面」として残り、ゆらゆら揺れる |
| `crossDepth + スクロール量` | 水中のスクロール。断面は画面の上へ抜けて消え、深くなるほど暗い |

「沈む」ボタン・「水面へ」ボタン・水中のドラッグスクロールは、すべてこの数字を動かしているだけ。
だから水平線と水面断面は**最初から最後まで同じ1本の線**として繋がって見える。逆再生（水面へ）も自動的に自然になる。

## 構成（依存なし・3ファイル）

- `index.html` — Canvas1枚＋細いUI（☰・目・記録を足す・沈む・チップ・水中ログ・水面へ・モーダル）
- `styles.css` — 静かな細い線のUI。ログは「カード」ではなく水中に浮かぶ小さな灯り
- `app.js` — 世界の全描画とカメラ。約700行

## Canvasが描いているもの

星・月と暈・対岸のシルエットと灯り／海のグラデーション・月光のきらめきの柱／
夜光虫（コアの渦のまわりに濃く分布、潜降フレア中は水面線へ吸い寄せられる）／
タップの波紋・記録が溶ける粒／下から見上げた水面（水の裏側・光だまり・波の背・ゴッドレイ3本）／
マリンスノー（潜降中は上へ流れて視差を作る）

## 記録

- タイトル自由入力＋分数（プリセット 10m/20m/30m/1h）
- localStorage キー `luminous-tide-v2`
- **初回はデモ記録7件**（参照画像と同じ並び）を自動で沈めてある。☰ → 「記録を消す」で空にできる

## 調整ツマミ（app.js）

- 潜降の長さ → `dive()` の `dur: 3.0`
- 断面が残る高さ → `LINE_TOP`
- 水平線の高さ → `resize()` の `H * 0.4`
- 暗くなる速さ → frame内 `dark = 0.5*dp + 0.45*extra`
- 夜光虫の数 → `seedWater()` の `n`
- コアの育ち → `drawCore()` の `total / 1500`
