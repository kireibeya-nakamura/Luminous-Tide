# Luminous Tide Mobile React Prototype

スマホ縦画面向けの、夜の海と努力記録のビジュアル・インタラクションプロトタイプです。

## 目的

ボタンを押して仮の作業記録を追加すると、画面下から水位が少し上がり、水面と青白い粒子が静かに反応します。

保存、認証、バックエンド、カレンダーはまだありません。まずは「記録したら水が満ちて少し報われる」感覚を検証するための1画面です。

## 起動

前提: Node.js 20.19以上、または22.12以上。

```bash
npm install
npm run dev
```

スマホで見る場合は、PCと同じWi-Fiにつないで、Viteが表示するNetwork URLを開きます。

## 構成

- `src/App.tsx`: アプリ入口
- `src/components/OceanScene.tsx`: 水位、ログ、波、粒子発生の状態管理
- `src/components/WaterSurface.tsx`: SVGの水面と水中表現
- `src/components/Particles.tsx`: Canvasの夜光虫風粒子
- `src/components/LogButton.tsx`: 記録ボタンと下部ドック
- `src/types.ts`: 共有型
- `src/styles.css`: スマホ縦画面向けの見た目

## 現在の挙動

- 初期水位は20%
- 記録ボタンで水位が5%上がる
- 最大水位は90%
- 水位はスプリング風に少し遅れて追従
- 旧作寄りのCanvas水面で、水面のゆらぎと粒子を描画
- 記録時に波が少し強くなる
- 記録時に水中へ青白い粒子が増え、数秒で消える
- 新作寄りのログカードを重ねて、直近ログを最大4件だけ控えめに表示
